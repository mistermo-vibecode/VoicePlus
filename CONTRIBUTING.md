# Contributing to VoicePlus

Thanks for your interest in contributing!

VoicePlus is a fork of the [Voice Audiobook Player](https://github.com/PaulWoitaschek/Voice) by Paul Woitaschek, licensed under GPL-3.0.

## How to contribute

- **Bug reports and feature ideas:** open an issue at <https://github.com/Mistermo/VoicePlus/issues>. Include your Android version, device model, app version, and reproduction steps where possible.
- **Code changes:** fork the repository, create a branch, make your changes, and open a pull request against `main`. Keep PRs focused; one concern per PR.
- **Discussion:** use GitHub Discussions for questions, ideas, and broader conversations.

## Development

See [`README.md`](README.md) for build instructions.

Run unit tests before submitting a PR:

```
./gradlew voiceUnitTest
```
